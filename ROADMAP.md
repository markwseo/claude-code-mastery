# 개발자 웹 포트폴리오 이력서 - 개발 로드맵

## 📋 프로젝트 개요
개인 포트폴리오 웹사이트를 통해 개발 경력, 기술 스택, 프로젝트 경험을 전시하는 반응형 이력서 사이트

## 🛠️ 기술 스택
- **HTML5**: 시맨틱 마크업
- **CSS3 / Tailwind CSS**: 현대적인 스타일링 및 반응형 디자인
- **JavaScript**: 인터랙티브 기능 및 동적 콘텐츠
